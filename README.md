# Placement Statistics Dashboard

## Project Overview

The **Placement Statistics Dashboard** is an interactive data analytics project that provides a comprehensive view of student placement performance, salary distribution, and hiring trends.

Built using **Excel, SQL, and Power BI**, this dashboard converts raw placement data into meaningful insights to support data-driven decisions for institutions and students.

## 📊 Detailed Insights

### 📈 Placement Trend (2020–2025)

* Placement rate increased steadily from **77.7% (2020)** to **79.4% (2025)**
* Indicates continuous improvement in placement performance

---

### 💰 Salary Distribution

* Highest concentration of students in **0–5 LPA (6348 students)**
* Followed by:

  * **5–10 LPA (5249 students)**
  * **10–15 LPA (5227 students)**
  * **15–20 LPA (3176 students)**
* Shows a **right-skewed salary distribution**

---

### 🎓 Placement Status

* ✅ **Placed Students:** ~16,000
* ❌ **Not Placed:** ~4,000
* Strong overall placement performance

---

### 🧩 Stream-wise Analysis

#### 📊 Total Students by Stream
